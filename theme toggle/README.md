# 🌓 Theme Toggle

A lightweight theme toggle project that switches between **Light Mode** and **Dark Mode** with a smooth UI and persistent settings using local storage.

---

## 🔗 Live Preview
🌍 [Live Preview](https://theme-toggle.surge.sh)
 

---

## ✨ Overview

This project demonstrates how to create a simple theme switcher using a checkbox input. The selected theme is saved in the browser so that it remains consistent even after refreshing the page.

---

## ⚙️ Functionality

- Toggle between light and dark background
- Animated switch button
- Theme state stored in `localStorage`
- Automatically loads saved theme on refresh

---

## 📁 Files Included

- `index.html` → Structure of the page  
- `style.css` → Styling and animation  
- `script.js` → Theme logic and storage
