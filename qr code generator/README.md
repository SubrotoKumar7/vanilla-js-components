# 🔳 QR Code Generator

A simple and fast web-based QR Code Generator built with HTML, CSS, and JavaScript.  
Generate and download QR codes instantly from any text or URL.

---

## 🚀 Live Demo
👉 [Live Link](free-qrcode.surge.sh)

---

## ✨ Features
- Generate QR codes from text or URLs
- Instant preview
- Download QR code as PNG image
- Simple and clean UI
- Toast alert for empty input

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QRCode.js Library

---



## ⚙️ How It Works
1. Enter text or a URL in the input field  
2. Click on **Generate QR**  
3. Your QR code will appear instantly  
4. Click **Download** to save it as an image
