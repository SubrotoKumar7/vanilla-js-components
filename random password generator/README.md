# 🔐 Random Password Generator

A simple and user-friendly **Random Password Generator** built using **HTML, CSS, and JavaScript**.  
This project allows users to generate strong random passwords and copy them to the clipboard with a single click.

---
## 🔗 Live Link
[Live Website](https://random-pass-generator.surge.sh)

---

## 🚀 Features

- 🎲 Generate a random password by clicking the **Generate** button
- 📋 Copy the generated password using the **Copy icon**
- ⚠️ Shows a warning if the copy button is clicked before generating a password  
  **"Please generate password first"**
- ✅ Displays a success message after copying the password  
  **"Password Copied!"**
- 📌 Copied password can be pasted anywhere (login forms, social media, emails, etc.)
- 🎨 Clean and responsive user interface

---

## 🛠️ Built With

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Logic and functionality  

---

## 📖 How It Works

1. Click the **Generate Password** button
2. A random password will appear on the screen
3. Click the **Copy icon** to copy the password
4. Paste the password wherever you need

> ⚠️ If you try to copy without generating a password first, a warning message will be shown.

---

## ⚠️ Messages Behavior

- **Warning Message:**  
  `Please generate password first`

- **Success Message:**  
  `Password Copied!`
